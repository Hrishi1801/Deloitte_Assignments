2. What is the difference between undefined and not defined in JavaScript?

The primary distinction between undefined and not defined is: A variable that has been declared but not assigned a value is undefined . A variable that has not been declared at all is not defined .

3. Difference between ==  and ===?

== is used for checking value, it will give true for both,
	10==10
	10=="10"

=== it is used for checking both data type and value,
	10==10		True
	10=="10"  	False

5. Difference between Client side JavaScript and Server side JavaScript?

server-side and client-side refer to the location where certain tasks or processes are carried out in a web application. Server-side processes are executed on the web server, while client-side processes are executed on the user's device.
